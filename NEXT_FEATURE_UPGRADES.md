# Next Feature Upgrades

## Recommended Next Steps

1. Add a map selector to the start screen so players can choose `Classic Front` or `Grand Front` manually instead of only switching by player count.
2. Show the selected map name and player capacity more prominently in the lobby before the host starts the match.
3. Add small map preview cards on the home screen so players can understand the size and style of each battlefield before creating a game.
4. Allow CPU games to use either map template intentionally, including the larger `Grand Front` layout.
5. Add map-specific balancing hooks so supply spacing, choke points, and starting room can be tuned per battlefield without rewriting core game setup logic.
